## Exercícios de Lógica e Algoritmos
### Manipulação de Números
- [Calcule a média de três números fornecidos pelo usuário.](/100-challenges/desafio1.mddesafio1.md)
- Verifique se um número é par ou ímpar.
- Converta uma temperatura de Celsius para Fahrenheit.
- Determine se um número é primo.Encontre o maior de três números fornecidos.
- Escreva uma função para calcular o fatorial de um número.
- Crie um algoritmo que exiba os números de 1 a 100, mas substitua os múltiplos de 3 por "Fizz" e os múltiplos de 5 por "Buzz".
- Converta um número de segundos para o formato HH:mm:ss.
- Crie uma função para verificar se um número é um palíndromo.
- Gere uma lista de números Fibonacci até um limite fornecido.

- Inverta uma string.

- Extraia o domínio de um endereço de e-mail.

- Valide se um CPF fornecido é válido.
- Formate um número de telefone para o padrão (XX) XXXX-XXXX.
